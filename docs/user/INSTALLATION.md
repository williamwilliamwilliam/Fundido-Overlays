# Fundido Overlays — Installation Guide

## System Requirements

- **Operating System:** Windows 10 or later (64-bit)
- **Graphics:** DirectX 11 compatible GPU (required for screen capture)
- **RAM:** 4 GB minimum (8 GB recommended if running alongside a game)

## Installation

1. Download the latest `Fundido-Overlays-Setup-x.x.x.exe` from the releases page.
2. Run the installer. It will install to your user's AppData directory — no admin privileges required.
3. Launch **Fundido Overlays** from the Start Menu or desktop shortcut.

## Uninstallation

Use **Add or Remove Programs** in Windows Settings, or run the uninstaller from the Start Menu.

## Data Location

Your configuration is stored at:

```
%APPDATA%/fundido-overlays/fundido-config.json
```

This file contains all your monitored regions, state rules, and overlay groups. You can back it up manually, or use the in-app Export feature to share individual pieces.
